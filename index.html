Co je to HnL SMP?

HnL SMP je první veřejný lifesteal smp server, který je zaměřen na zábavu hráčů. Připojit na server se dá z verze 1.20.4 na Java Edici, pro připojení na server potřebuješ být na našem discordu a verifikovat se (/nick). 

Mnoha updatů

U nás nové možnosti, gamemody, itemy a mnoho dalšího přidáváme často a dáváme členům discordu o nich vědět předem! 

Vlastní gamemody!

Na serveru máme vlastně udělané pluginy jako např. LifeSteal, ovšem máme také i jiné třeba SlimeFun. 

Velmi aktivní A-Team

Náš admin team na serveru i na discordu je velice aktivní a milý. Ze vším vám pomůže. 

Jak se připojit?

Níže můžeš vidět guide jak se připojit na server.

1

Přidej si server

Připoj se na náš discord a verifikuj se.
Zapni minecraft.
2

Připoj se!

V minecraftu klikni na Multiplayer -> Add Server
a zadej mc.hnlsmp.pro a připoj se!
3

Hotovo!
